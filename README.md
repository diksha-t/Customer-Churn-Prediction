# Customer Churn Prediction & Business Insights

## Project Overview

This project aims to predict customer churn for a subscription-based service using ML. In addition to predicting whether a customer is likely to churn, the project identifies the key factors influencing churn and provides business insights that can help improve customer retention.

The final solution will be deployed as an interactive Streamlit dashboard.

---

## Tech Stack
- Python
- scikit-learn
- Seaborn
- Streamlit
